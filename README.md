Проект выполнен на базе телеграмм бота

Бот информационного характера дающего информацию про глобальное потепление

Команды бота:
/info - что такое глобальное потепление
/fact - факт о климате
/help_earth - как помочь планете
/img - тематическое изображение
Бот создан для осведомления большого количества людей про проблему глобального потепления, ведь эта, на первый взгляд маленькая проблема может перерасти в будщем в катострофический катаклизм для нашей планеты, ввиду повышения средней температуры земли с каждым годом, которое влечет за собой повышения уровня моря, большие изменения климата в тропических и суб-тропических регионах также вымирание некоторых видов птиц, молльюсков, бактерий, и млекопитающих.
